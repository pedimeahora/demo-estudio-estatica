# Estudio Jurídico Giangreco

Sitio web profesional para un estudio jurídico especializado en accidentes de tránsito, derecho laboral, familia y jubilaciones.

## 🚀 Inicio Rápido

### 1. Instalar dependencias
```bash
npm install
```

### 2. Desarrollar localmente
```bash
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.

### 3. Compilar para producción
```bash
npm run build
```

### 4. Desplegar en GitHub Pages
```bash
npm run deploy
```

## 📁 Estructura del Proyecto

```
src/
├── components/       # Componentes React
│   ├── Navbar.tsx
│   ├── Hero.tsx
│   ├── Services.tsx
│   ├── WhyUs.tsx
│   ├── Contact.tsx
│   ├── Footer.tsx
│   └── WhatsappFab.tsx
├── routes/          # Rutas de TanStack Router
│   ├── __root.tsx
│   └── index.tsx
├── lib/             # Funciones utilitarias
├── hooks/           # Custom hooks
├── main.tsx         # Entry point
├── index.css        # Estilos globales
└── routeTree.gen.ts # Generado por TanStack Router
```

## 🛠 Tecnologías

- **React 19** - Framework UI
- **Vite** - Build tool
- **TanStack Router** - Enrutamiento
- **TanStack Query** - Gestión de estado
- **Tailwind CSS** - Estilos
- **TypeScript** - Type safety
- **Lucide React** - Iconos
